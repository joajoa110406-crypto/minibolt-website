# MiniBolt 스크류 판매 웹사이트

산업용 마이크로 스크류 전문 판매 웹사이트

## 📁 파일 구조

```
minibolt-website/
├── index.html              # 랜딩페이지 (메인 페이지)
├── products.html           # 제품 선택 페이지
├── products.js             # 제품 데이터 (JavaScript 형식)
├── 재품단가.json           # 제품 데이터 원본 (JSON 형식)
└── README.md              # 프로젝트 설명 (이 파일)
```

## 🚀 사용 방법

### 방법 1: 브라우저에서 직접 열기 (간단)
1. `index.html` 파일을 더블클릭
2. 브라우저에서 바로 열립니다
3. 모든 기능이 정상 작동합니다

### 방법 2: 로컬 서버 실행 (권장)
```bash
# 이 폴더에서 실행
python -m http.server 8000

# 브라우저에서 접속
# http://localhost:8000
```

## 📋 페이지 구성

### 1. 랜딩페이지 (index.html)
- **히어로 섹션**: 사이트 소개
- **제품 카테고리**: 4가지 스크류 타입 카드
  - 마이크로스크류/평머리
  - 바인드헤드
  - 팬헤드
  - 플랫헤드
- **가격 안내**
- **문의 정보**

### 2. 제품 선택 페이지 (products.html)
- **카테고리 탭**: 4개 카테고리 전환
- **필터 기능**:
  - 직경 (M1, M2, M2.5 등)
  - 길이 (2mm, 3mm, 4mm 등)
  - 색상/코팅 (블랙, 니켈 등)
- **제품 카드**:
  - 제품 정보
  - 재고 상태
  - 가격 (단가, 100개, 1000개)
  - 수량 선택
  - 장바구니 담기
- **장바구니 기능**

## 📊 제품 데이터

### products.js
- 제품 데이터를 JavaScript 변수로 저장
- 브라우저에서 직접 로드 가능
- 서버 없이도 작동

### 재품단가.json
- 원본 제품 데이터 (313KB)
- 필요 시 수정 후 products.js로 변환

**JSON → JS 변환 방법:**
```bash
echo "const productData = " > products.js
cat 재품단가.json >> products.js
echo ";" >> products.js
```

## 🐛 문제 해결

### Q: 제품 정보가 안 보여요
A: `products.js` 파일이 같은 폴더에 있는지 확인하세요

### Q: 이미지가 안 보여요
A: 이미지가 base64로 인코딩되어 HTML에 포함되어 있어 별도 파일 불필요

### Q: 링크가 작동 안 해요
A: 모든 HTML 파일이 같은 폴더에 있는지 확인하세요
