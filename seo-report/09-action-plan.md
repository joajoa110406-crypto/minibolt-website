## 1. 경쟁 우위 분석: 미니볼트 vs 나사코리아

### 1.1 핵심 차별점 요약

| 비교 항목 | 미니볼트 (MiniBolt) | 나사코리아 (경쟁사) | 우위 |
|---|---|---|---|
| **판매 구조** | 제조사 직접판매 (성원특수금속) | 리셀러/유통업체 | **미니볼트** |
| **소량 주문** | 100개 단위, 3,300원부터 | 소량 불가 (대량 위주) | **미니볼트** |
| **제조 경험** | 39년 (1987년~) | 유통 이력만 보유 | **미니볼트** |
| **제품 수** | 833종 SKU | 다양하나 직접 제조 아님 | 비슷 |
| **가격 경쟁력** | 중간 마진 없음 (공장 직판) | 유통 마진 포함 | **미니볼트** |
| **배송 속도** | 국내 제조, 당일~익일 출고 | 일반 배송 | **미니볼트** |
| **기술 상담** | 제조사가 직접 상담, 맞춤 제작 가능 | 일반 CS 수준 | **미니볼트** |
| **환경 인증** | ROHS 6종 준수 | 미확인 | **미니볼트** |

### 1.2 미니볼트 핵심 USP (Unique Selling Proposition) 3가지

1. **"제조사 직판"** -- 성원특수금속이 직접 만들고 직접 팝니다. 중간 유통 마진이 없어 가격 경쟁력이 있습니다.
2. **"소량 구매 가능"** -- 100개 단위부터 주문 가능. 개인 개발자, 메이커, 수리 업체도 부담 없이 구매할 수 있습니다.
3. **"39년 제조 전문성"** -- 1987년 창립 이후 마이크로 스크류만 전문 제조. 규격 상담과 맞춤 제작이 가능합니다.

### 1.3 SEO 관점에서의 경쟁 우위 활용 전략

나사코리아 대비 미니볼트가 SEO에서 우위를 점할 수 있는 영역:

- **롱테일 키워드**: "M2 나사 100개", "M1.6 나사 소량 구매", "안경 나사 소량" 등 소량 구매 의도 키워드
- **신뢰도 콘텐츠**: 제조사이므로 제조 과정, 품질 관리, 기술 사양 등 전문 콘텐츠 생산 가능
- **가격 투명성**: 리셀러가 제공하기 어려운 제조원가 기반 가격표 공개
- **기술 콘텐츠**: 나사 선택 가이드, 규격 비교표, 토크 사양 등 제조사만 쓸 수 있는 콘텐츠

## 2. USP 메시징 일관성 점검

### 2.1 페이지별 메시징 현황

각 페이지에서 5가지 핵심 메시지가 얼마나 일관되게 전달되는지 점검했습니다.

| 페이지 | 제조사 직판 | 소량 구매 가능 | 39년/1987년 | 미니볼트 브랜드 | 성원특수금속 |
|---|:---:|:---:|:---:|:---:|:---:|
| **layout.tsx** (사이트 전체 메타) | O (title, description, OG, JSON-LD) | O (description) | O (description) | O (title) | O (JSON-LD) |
| **page.tsx** (메인 랜딩) | O (제조사 직접 판매, 중간 마진 없음) | O (100개부터 주문 가능) | O (39년 제조 업력) | O (Mini Bolt) | O (성원특수금속) |
| **products/layout.tsx** (제품 목록 메타) | O (제조사 직판) | O (100개부터) | X (누락) | O (미니볼트) | X (누락) |
| **products/page.tsx** (제품 목록 본문) | X (누락) | X (누락) | X (누락) | X (누락) | X (누락) |
| **products/[id]/page.tsx** (제품 상세) | O (직접판매) | O (소량 100개부터) | O (39년) | O (미니볼트) | O (성원특수금속) |
| **company/page.tsx** (회사소개) | O (제조사 직접 판매) | O (100개 소량 주문) | O (39년, 1987년) | O (미니볼트) | O (성원특수금속) |

| **Header.tsx** (네비게이션) | X (누락) | X (누락) | X (누락) | O (Mini Bolt 로고) | X (누락) |
| **Footer.tsx** (하단) | X (부분 - "전문" 언급) | X (누락) | X (누락) | O (미니볼트) | O (성원특수금속) |

### 2.2 메시징 일관성 평가

**잘 되어 있는 부분:**
- 메인 페이지(page.tsx): 모든 USP가 체계적으로 전달됨 (강점 카드, 통계, 가격 섹션)
- layout.tsx 메타데이터: title, description, OpenGraph, JSON-LD 모두 우수
- 제품 상세 페이지: meta description, JSON-LD, SEO 설명 섹션까지 완비
- 회사소개 페이지: 모든 USP가 명확하게 전달됨

**개선이 필요한 부분:**
- **제품 목록 본문(products/page.tsx)**: 가장 트래픽이 많을 페이지인데 USP 메시지가 전혀 없음. 히어로 섹션에 "마이크로스크류 선택"만 표시
- **문의하기 페이지(contact/page.tsx)**: 제조사 직판 신뢰 메시지 부재
- **결제 페이지(checkout/page.tsx)**: 제조사 직판 안심 메시지 부재
- **헤더(Header.tsx)**: 로고 옆에 "제조사 직판" 태그라인 없음
- **푸터(Footer.tsx)**: "제조사 직접판매", "39년 제조 경험" 등 핵심 메시지 누락

### 2.3 심각도 순위 (우선 수정 필요)

1. **[높음]** `products/page.tsx` -- 제품 히어로에 USP 메시지 추가 필요
2. **[높음]** `Header.tsx` -- 로고 옆 또는 하단에 "제조사 직판" 태그라인 추가
3. **[중간]** `Footer.tsx` -- "39년 제조사 직접판매" 메시지 보강
4. **[중간]** `contact/page.tsx` -- 문의 페이지에 신뢰도 메시지 추가
5. **[낮음]** `checkout/page.tsx` -- 결제 시 안심 메시지 (SEO 영향 적음, 전환율 영향)

| **제조 과정 소개** | 없음 | 높음 - 신뢰도 구축, 제조사 증명 | 1순위 |

### 4.2 핵심 콘텐츠 갭 상세

#### (1) 비교 페이지: "제조사 직판 vs 리셀러"

지금은 메인 페이지 강점 카드에서만 간접적으로 전달. 별도 랜딩 페이지로 직접 비교표 제공 필요.

**포함 내용:**
- 가격 비교표 (동일 규격 기준)
- 품질 보증 차이 (제조사 직접 vs 유통)
- 기술 상담 가능 여부
- 맞춤 제작 가능 여부
- 납기 차이

**타겟 키워드:** "마이크로 나사 저렴하게", "나사 직접 구매", "나사 공장 직판"

#### (2) 용도별 가이드 페이지

현재 메타 키워드에는 용도별 키워드(안경나사, 노트북나사 등)가 포함되어 있지만, 실제 콘텐츠 페이지는 없음.

**필요한 가이드:**
- `/guides/glasses-screw` -- 안경 나사 선택 가이드 (M1.2~M1.6)
- `/guides/laptop-screw` -- 노트북 나사 규격 가이드
- `/guides/ssd-screw` -- SSD/NVMe 고정 나사 가이드
- `/guides/camera-screw` -- 카메라 마운트 나사 가이드
- `/guides/raspberry-pi` -- 라즈베리파이 나사 가이드
- `/guides/drone-screw` -- 드론 조립용 나사 가이드

#### (3) FAQ 페이지

현재 전혀 없음. FAQ Schema를 적용하면 Google 검색결과에 풍부한 스니펫으로 표시 가능.

**예상 FAQ 항목:**
- 최소 주문 수량은? (100개)
- 배송은 얼마나 걸리나요? (1~2일)
- 맞춤 제작이 가능한가요?
- VAT가 포함된 가격인가요?
- 반품/교환은 어떻게 하나요?
- 나사 규격을 모르면 어떻게 하나요?
- 대량 구매 할인이 있나요?
