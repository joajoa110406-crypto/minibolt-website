## 1. 현황 분석

### 1.1 기존 상태 (변경 전)

| 항목 | 상태 | 비고 |

| robots.txt Yeti 허용 | 부분적 | `User-agent: *`로 모든 봇 허용하나, Yeti 전용 규칙 없음 |
| OG 태그 (메인) | 기본 | og:title, og:description, og:image 존재 |

### 1.2 핵심 문제점

**가장 큰 문제: 제품 목록 페이지 (`/products`) SSR 미지원**

`src/app/products/page.tsx` 파일은 `'use client'` 지시어를 사용하며, `mounted` 상태가 `true`가 될 때까지 스켈레톤 UI만 렌더링합니다. 이는 한글 데이터의 ByteString 이슈를 방지하기 위한 것이지만, 네이버 Yeti 크롤러는 JavaScript를 제한적으로 실행하므로 제품 목록을 전혀 인덱싱하지 못할 수 있습니다.

```
// products/page.tsx 현재 구조
'use client';
// ...
if (!mounted) return <스켈레톤 UI>; // Yeti 크롤러가 보는 내용
```

**다행히 개별 제품 상세 페이지 (`/products/[id]`)는 Server Component로 구현되어 있고, ISR(3600초)을 사용하므로 네이버 크롤링에 문제가 없습니다.**

Allow: /

Allow: /products

## 5. 발견된 문제점 및 개선 권장사항

### 5.1 치명적 문제: 제품 목록 페이지 CSR

**문제:**
`/products` 페이지가 `'use client'`로 선언되어 있고, `mounted` 상태로 SSR을 차단합니다. 네이버 Yeti 크롤러는 JavaScript 실행이 제한적이므로, 이 페이지의 제품 목록을 인덱싱하지 못합니다.

**영향:**
- 네이버에서 "마이크로 스크류", "소형 나사" 등으로 검색 시 `/products` 페이지가 노출되지 않을 수 있음
- 다행히 개별 제품 상세 페이지(`/products/[id]`)는 SSR이므로 인덱싱 가능

**권장 해결 방안:**
1. **Server Component로 전환:** 제품 목록을 서버에서 렌더링하고, 필터/검색만 Client Component로 분리
2. **Hybrid 접근:** 초기 제품 목록(첫 50개)은 서버에서 렌더링하고, 이후 필터/검색은 클라이언트에서 처리
3. **한글 ByteString 이슈 해결:** Next.js의 `encodeURIComponent` 또는 `Buffer` 활용으로 한글 데이터 SSR 가능하도록 개선

```tsx
// 권장 구조 (예시)
// products/page.tsx (Server Component)
export default async function ProductsPage() {
  const products = await getProducts(); // 서버에서 데이터 로드
  return (
    <div>
      <ProductFilters /> {/* 'use client' */}
      <ProductGrid initialProducts={products} /> {/* 'use client', 초기 데이터는 서버에서 */}
    </div>
  );
}
```

**현재:** 제품 카테고리 이미지(image-1~4.png)를 OG 이미지로 사용 중

## 6. 네이버 블로그 포스트 초안 (5편)

---

### 포스트 1: 마이크로나사 규격 완벽 가이드 - M1.2부터 M3까지

**키워드:** 마이크로나사 규격, 소형나사 규격, M1.2 나사, M1.4 나사, M1.6 나사, M2 나사, M2.5 나사, M3 나사

---

마이크로나사는 전자기기, 정밀기기, 안경, 시계 등 다양한 분야에서 사용되는 아주 작은 나사입니다. 하지만 규격이 다양하고 용도도 제각각이라 어떤 나사를 선택해야 할지 막막할 때가 많습니다. 오늘은 가장 많이 사용되는 M1.2부터 M3까지의 마이크로나사 규격을 하나씩 살펴보겠습니다.

**M1.2 나사 - 가장 작은 정밀 나사**

M1.2는 직경이 1.2mm인 초소형 나사입니다. 주로 안경 힌지, 초소형 전자부품, 시계 내부 부품 고정에 사용됩니다. 크기가 매우 작기 때문에 정밀 드라이버가 필수이며, 스테인리스 스틸 재질이 많이 쓰입니다. 안경을 수리할 때 가장 많이 찾는 규격이기도 합니다.

**M1.4 나사 - 안경/시계 표준 규격**

M1.4는 안경테와 시계에 가장 많이 사용되는 규격입니다. M1.2보다 약간 큰 1.4mm 직경으로, 안경 다리(템플)를 고정하는 데 주로 쓰입니다. 안경원에서 수리할 때 사용하는 나사가 대부분 M1.4입니다. 길이는 2mm에서 6mm까지 다양합니다.

**M1.6 나사 - 소형 전자기기의 핵심**

M1.6은 스마트폰 내부, 소형 센서, 미니 드론의 프레임 고정 등에 사용됩니다. M1.4보다 약간 큰 규격으로 내구성이 더 높아, 진동이 있는 환경에서도 안정적입니다. 3D프린터의 핫엔드 고정 나사로도 많이 쓰입니다.

**M2 나사 - 가장 범용적인 마이크로 규격**

M2는 마이크로나사 중 가장 많이 사용되는 규격입니다. 라즈베리파이 고정, SSD 장착, 노트북 내부 부품 고정, 드론 모터 마운트, RC카 조립 등 용도가 매우 다양합니다. 길이는 3mm부터 20mm까지 폭넓게 있으며, 머신스크류(M/C)와 태핑스크류(T/C) 모두 생산됩니다.

**M2.5 나사 - 노트북/HDD의 표준**

M2.5는 노트북 HDD, 광학드라이브, 일부 데스크톱 SSD 브래킷 고정에 사용되는 규격입니다. 일반 데스크톱의 M3 나사와 노트북용 M2 나사의 중간 규격으로, 적당한 강도와 크기를 가지고 있습니다.

**M3 나사 - 마이크로와 표준의 경계**

M3는 마이크로나사와 일반 나사의 경계에 있는 규격입니다. 데스크톱 PC의 메인보드 고정, 팬 장착, 3D프린터 프레임 조립 등 소형 기계 조립에 널리 사용됩니다. 강도가 높아 구조물 고정에도 적합합니다.

**어떤 규격을 선택해야 할까?**

| 용도 | 추천 규격 |
|------|-----------|
| 안경 수리 | M1.2, M1.4 |
| 시계 수리 | M1.2, M1.4 |
| 스마트폰/태블릿 | M1.4, M1.6 |
| 라즈베리파이/아두이노 | M2, M2.5 |
| SSD 장착 | M2 |
| 노트북 수리 | M2, M2.5 |
| 드론 조립 | M2, M3 |
| 3D프린터 | M2, M3 |

마이크로나사 규격 선택이 어렵다면, 미니볼트(minibolt.co.kr)에서 규격별로 소량 구매가 가능합니다. 100개 단위로 주문할 수 있어 필요한 만큼만 구매하면 됩니다. 39년 제조 경험의 성원특수금속에서 직접 생산하고 판매하므로 품질도 보장됩니다.

---

### 포스트 2: 3D프린터 나사 선택법 - 정밀 조립에 필요한 마이크로 스크류

**키워드:** 3D프린터 나사, 3D프린터 조립 나사, M2 나사, M3 나사, 3D프린터 핫엔드 나사, 3D프린터 부품

---

3D프린터를 조립하거나 유지보수할 때, 의외로 많은 종류의 나사가 필요합니다. 특히 정밀한 출력 품질을 위해서는 적절한 규격의 나사를 선택하는 것이 중요합니다. 오늘은 3D프린터에 사용되는 나사 규격과 선택 기준에 대해 알아보겠습니다.

**3D프린터에 사용되는 주요 나사 규격**

3D프린터에는 주로 M2, M2.5, M3 규격의 나사가 사용됩니다. 부품에 따라 필요한 규격이 다릅니다.

- 핫엔드 고정: M3 또는 M2.5 (열에 노출되므로 스테인리스 스틸 권장)
- 스텝모터 장착: M3 (4개씩, 총 12~16개 필요)
- 리니어 레일/베어링 고정: M2, M3
- 팬 장착: M2, M2.5, M3 (팬 크기에 따라 다름)
- PCB/컨트롤 보드 고정: M2.5, M3
- 프레임 조립: M3, M4
- 엔드스톱 센서 고정: M2
- 벨트 텐셔너: M3

**평머리 vs 접시머리 선택 기준**

3D프린터에서 나사 머리 형태 선택은 생각보다 중요합니다.

평머리(팬헤드/바인드헤드) 나사는 머리가 둥글게 올라와 있어 조임 상태를 시각적으로 확인하기 쉽습니다. 일반적인 조립에 가장 많이 사용됩니다. 스텝모터, 팬, PCB 고정 등 대부분의 부위에 적합합니다.

접시머리(플랫헤드/카운터싱크) 나사는 나사 머리가 표면과 같은 높이로 매립됩니다. 캐리지 플레이트처럼 평평한 면이 필요한 곳에 사용합니다. 핫엔드 주변에서 간섭을 줄일 때도 유용합니다. 단, 카운터싱크 가공이 된 부품에만 사용할 수 있습니다.

**머신스크류 vs 태핑스크류**

머신스크류(M/C)는 미리 탭이 가공된 금속 부품에 사용합니다. 알루미늄 프레임, 스틸 브래킷 등에 적합하며, 반복 체결/해제가 가능합니다.

태핑스크류(T/C)는 나사 자체가 나사산을 만들면서 들어갑니다. 플라스틱 부품이나 3D 프린팅된 부품에 직접 체결할 때 사용합니다. 처음 체결 시 나사산이 형성되므로, 같은 구멍에 너무 여러 번 체결/해제하면 나사산이 손상될 수 있습니다.

**색상 선택: 블랙 vs 니켈**

블랙 나사는 검은색 프레임의 3D프린터에 어울리며, 눈에 띄지 않는 깔끔한 마감을 제공합니다. Voron, Ender 시리즈 등 검은색 프린터에 추천합니다.

니켈 나사는 은색 계열로, 밝은 색상의 프린터나 시각적으로 나사 위치를 확인해야 할 때 유용합니다.

**3D프린터 나사 구매 팁**

3D프린터 조립 시에는 여러 규격의 나사가 소량씩 필요합니다. 미니볼트(minibolt.co.kr)에서는 M2, M2.5, M3 나사를 각각 100개 단위로 구매할 수 있어, 다양한 규격을 필요한 만큼만 구입할 수 있습니다. 39년 제조 경험의 성원특수금속에서 직접 생산하는 스테인리스 스틸 나사이므로 품질이 균일하고, 열에 강합니다.

특히 3D프린터 유저에게 인기 있는 규격:
- M2x6 블랙 평머리 (팬 고정용)
- M3x8 블랙 평머리 (스텝모터용)
- M3x10 블랙 평머리 (프레임 조립용)
- M2.5x6 니켈 팬헤드 (PCB 고정용)

### 포스트 3: 안경 나사 교체 방법 - 집에서 쉽게 수리하기

**키워드:** 안경 나사, 안경나사 교체, 안경테 나사, 안경 수리, M1.2 나사, M1.4 나사, 안경나사 규격

---

안경을 오래 사용하다 보면 나사가 풀리거나 빠지는 경우가 흔합니다. 안경원에 가면 간단히 해결할 수 있지만, 시간이 없거나 근처에 안경원이 없을 때는 집에서 직접 교체할 수 있습니다. 생각보다 어렵지 않으니 따라해 보세요.

**안경에 사용되는 나사 규격**

안경에는 주로 M1.2와 M1.4 규격의 마이크로나사가 사용됩니다.

- 안경 다리(템플) 힌지: 주로 M1.4 x 3~5mm
- 코패드 연결부: 주로 M1.2 x 2~3mm
- 렌즈 고정 (반무테): M1.4 x 4~6mm
- 브릿지 연결부: M1.2 x 2~4mm

대부분의 안경 나사는 M1.4 규격이며, 길이 3~4mm가 가장 범용적입니다. 안경 나사를 교체할 때 정확한 규격을 모른다면, M1.4 x 4mm를 먼저 시도해 보세요.

**준비물**

1. 안경용 정밀 드라이버 (십자 또는 일자, 1.0~1.5mm)
2. 교체용 마이크로나사 (M1.2 또는 M1.4)
3. 밝은 조명
4. 흰색 천이나 수건 (나사를 떨어뜨렸을 때 찾기 쉬움)
5. 핀셋 (선택사항이지만 강력 추천)

**교체 방법 Step-by-Step**

Step 1. 작업 환경 준비
밝은 곳에서 흰색 천 위에 작업합니다. 마이크로나사는 매우 작아서 떨어뜨리면 찾기 어렵습니다. 바닥에 카펫이 깔려 있다면 더욱 주의하세요.

Step 2. 기존 나사 확인
빠진 나사가 어디에 있었는지 확인합니다. 안경 다리 힌지, 코패드, 렌즈 고정 부분 중 어디인지에 따라 필요한 나사 규격이 다릅니다.

Step 3. 나사 구멍 정렬
나사가 빠진 부분의 구멍을 정확히 맞춥니다. 안경 다리 힌지의 경우, 다리를 약간 접은 상태에서 구멍을 맞추면 쉽습니다.

Step 4. 나사 삽입
핀셋으로 나사를 잡고 구멍에 넣습니다. 맨손으로 하면 나사가 작아서 놓치기 쉬우므로 핀셋 사용을 강력 추천합니다.

Step 5. 드라이버로 조임
정밀 드라이버로 나사를 시계방향으로 돌려 조입니다. 너무 세게 조이면 나사산이 손상되거나 안경테가 파손될 수 있으니, 적당한 힘으로 조여주세요. 나사가 빠지지 않을 정도로만 조이면 됩니다.

Step 6. 풀림 방지 처리 (선택)
나사가 자주 풀리는 경우, 투명 매니큐어를 나사 머리에 한 방울 발라주면 풀림을 방지할 수 있습니다.

**자주 묻는 질문**

Q. 나사 규격을 모르겠어요.
A. 대부분의 안경 나사는 M1.4 규격입니다. M1.4 x 3mm, M1.4 x 4mm 두 가지를 준비하면 대부분 대응 가능합니다.

Q. 나사가 너무 길면 어떻게 되나요?
A. 나사가 돌출되어 피부에 닿을 수 있습니다. 이 경우 더 짧은 나사를 사용하세요.

Q. 정밀 드라이버가 없어요.
A. 안경 수리 키트를 구매하거나, 다이소 등에서 정밀 드라이버 세트를 구입할 수 있습니다.

안경 나사는 미니볼트(minibolt.co.kr)에서 M1.2, M1.4 규격을 100개 단위로 소량 구매할 수 있습니다. 한 번 구매해두면 안경 나사가 빠질 때마다 바로 교체할 수 있어 편리합니다. 100개에 3,000원(VAT 별도)으로 부담 없는 가격입니다.

### 포스트 4: 드론 조립용 나사 가이드 - 진동에 강한 정밀나사 선택

**키워드:** 드론 나사, 드론 조립 나사, 드론 프레임 나사, M2 나사, M3 나사, 드론 모터 나사, 진동에 강한 나사

---

드론을 직접 조립하거나 수리할 때 나사 선택은 비행 안전과 직결됩니다. 드론은 비행 중 강한 진동과 충격에 노출되기 때문에, 적절한 규격과 재질의 나사를 사용해야 합니다. 잘못된 나사를 사용하면 비행 중 부품이 분리되어 추락할 수 있습니다.

**드론에 사용되는 주요 나사 규격**

드론의 크기(휠베이스)에 따라 사용되는 나사 규격이 다릅니다.

소형 드론 (65~150mm급, 타이니훕/마이크로):
- 모터 장착: M1.6 x 4~5mm 또는 M2 x 4~6mm
- FC/ESC 스택: M2 x 6~8mm (나일론 스탠드오프 포함)
- 카메라 고정: M2 x 5~6mm

중형 드론 (150~250mm급, 5인치):
- 모터 장착: M3 x 6~8mm (대부분 M3)
- FC/ESC 스택: M3 x 20~30mm (스탠드오프)
- 프레임 조립: M3 x 6~12mm
- 카메라 마운트: M2 x 6~8mm
- VTX 안테나 고정: M2 x 8~10mm

대형 드론 (250mm 이상, 7~10인치):
- 모터 장착: M3 x 8~10mm
- 프레임 조립: M3, M4
- 프롭 어댑터: 규격 다양

**진동에 강한 나사 선택 기준**

드론에서 나사가 풀리는 주요 원인은 모터 진동입니다. 진동에 강한 나사를 선택하려면 다음 사항을 고려하세요.

첫째, 재질을 확인하세요. 스테인리스 스틸 나사는 강도가 높고 부식에 강합니다. 야외 비행이 많은 경우 스테인리스 스틸이 필수입니다. 미니볼트의 나사는 모두 스테인리스 스틸 재질로 제작됩니다.

둘째, 나사 타입을 선택하세요. 금속 프레임(알루미늄, 카본)에는 머신스크류(M/C)를 사용합니다. 탭이 가공된 구멍에 체결하므로 반복 분해/조립이 가능합니다. 플라스틱 부품(카메라 마운트, 캐노피)에는 태핑스크류(T/C)를 사용하면 나사산을 직접 형성하여 단단하게 고정됩니다.

셋째, 적절한 길이를 선택하세요. 나사가 너무 짧으면 고정력이 부족하고, 너무 길면 다른 부품과 간섭할 수 있습니다. 최소 3개 이상의 나사산이 물려야 안정적입니다.

**풀림 방지 팁**

1. 나사잠금제(Loctite) 사용: 중강도(파란색) 나사잠금제를 나사산에 한 방울 발라 체결하면 진동에 의한 풀림을 효과적으로 방지합니다.
2. 스프링 와셔 사용: M3 나사에 스프링 와셔를 추가하면 풀림 방지에 도움이 됩니다.
3. 정기 점검: 비행 전후로 나사 조임 상태를 확인하는 습관을 들이세요.

**드론 조립 시 나사 수량 계산**

5인치 드론 한 대 기준 필요한 나사 수량:
- M3 x 6mm (모터 장착): 16개 (4개 x 4모터)
- M3 x 8mm (프레임 조립): 8~12개
- M3 x 20mm 스탠드오프: 4개
- M2 x 6mm (카메라, VTX): 8~10개
- 여분 포함 총: 약 50~60개

미니볼트(minibolt.co.kr)에서는 M2, M3 나사를 100개 단위로 구매할 수 있어 드론 한 대 조립에 필요한 수량을 여유 있게 확보할 수 있습니다. 스테인리스 스틸 재질이므로 야외 비행에서도 부식 걱정이 없습니다. 100개 3,000원부터로 합리적인 가격입니다.

---

### 포스트 5: 나사 직구 vs 국내 제조사 직판 - 마이크로나사 구매 가이드

**키워드:** 마이크로나사 구매, 소형나사 구매, 나사 소량구매, 나사 직구, 나사 제조사, 나사 판매, M2 나사 구매

---

마이크로나사를 구매하려고 하면 선택지가 생각보다 많지 않습니다. 해외 직구, 국내 리셀러, 그리고 제조사 직판. 각각의 장단점을 비교하여 가장 효율적인 구매 방법을 알아보겠습니다.

**해외 직구의 현실**

AliExpress, Amazon 등에서 마이크로나사를 직구하는 분들이 있습니다. 가격이 저렴해 보이지만, 실제로는 여러 가지 문제가 있습니다.

배송 시간이 깁니다. 중국발 배송은 최소 2~3주, 길면 한 달 이상 소요됩니다. 프로젝트 일정이 촉박할 때 치명적입니다. 급하게 필요한 나사가 한 달 뒤에 도착하면 의미가 없습니다.

품질이 불균일합니다. 저렴한 해외 나사는 재질이 불분명하거나 도금 품질이 떨어지는 경우가 많습니다. 나사산 정밀도가 낮아 체결이 어렵거나, 쉽게 녹이 슬기도 합니다. 특히 마이크로나사는 정밀도가 중요한데, 저품질 나사는 나사산이 맞지 않아 부품을 손상시킬 수 있습니다.

규격 혼동이 있습니다. 해외 판매자의 규격 표기가 부정확한 경우가 있습니다. M2를 주문했는데 실제로는 M2.3이 오거나, 길이가 표기와 다른 경우도 있습니다.

소량 구매가 어렵습니다. 대부분 최소 500개 이상 단위로 판매하며, 여러 규격을 소량씩 구매하기 어렵습니다.

**국내 리셀러의 한계**

국내 온라인 쇼핑몰(쿠팡, 네이버 등)에서 마이크로나사를 검색하면 리셀러(유통업체)의 제품이 나옵니다. 품질은 대체로 괜찮지만 다음과 같은 단점이 있습니다.

가격이 비쌉니다. 제조사에서 구매한 제품을 재포장하여 판매하므로, 중간 유통 마진이 추가됩니다. 동일 규격의 나사가 제조사 직판 대비 2~5배 비싼 경우도 흔합니다.

규격 선택이 제한적입니다. 리셀러는 인기 규격 위주로만 재고를 보유합니다. M2 x 6mm은 있지만 M2 x 4mm은 없는 식입니다. 필요한 정확한 규격을 찾기 어려울 수 있습니다.

규격 상담이 어렵습니다. 리셀러는 나사 전문가가 아닌 경우가 많아, 정확한 규격 상담을 받기 어렵습니다.

**제조사 직판의 장점 - 미니볼트(MiniBolt)**

미니볼트는 39년 제조 경험의 성원특수금속에서 직접 운영하는 마이크로나사 전문 온라인몰입니다. 제조사가 직접 판매하기 때문에 다음과 같은 장점이 있습니다.

합리적인 가격입니다. 중간 유통 단계가 없으므로 합리적인 가격에 구매할 수 있습니다. 100개 단위로 3,000원(VAT 별도)부터 시작하며, 대량 구매 시 추가 할인이 적용됩니다.

833종의 풍부한 규격입니다. M1.2부터 M4까지, 다양한 직경/길이/헤드타입/색상 조합으로 833종의 제품을 보유하고 있습니다. 정확히 필요한 규격을 찾을 수 있습니다.

소량 구매가 가능합니다. 100개 단위로 주문할 수 있어, 개인 개발자나 소규모 프로젝트에도 부담 없이 구매할 수 있습니다. 여러 규격을 100개씩 소량으로 주문하는 것도 가능합니다.

빠른 배송입니다. 국내 제조/국내 발송이므로 주문 후 1~2일 내 수령 가능합니다. 해외 직구처럼 2~3주를 기다릴 필요가 없습니다.

정확한 규격 상담이 가능합니다. 제조사가 직접 운영하므로 정확한 규격 상담이 가능합니다. 어떤 규격이 필요한지 모를 때 용도를 설명하면 적합한 제품을 추천받을 수 있습니다.

균일한 품질 보증입니다. 모든 제품이 스테인리스 스틸 재질이며, 자체 공장에서 생산하므로 품질이 균일합니다. ROHS 6종 환경규제를 준수하여 유해물질 걱정도 없습니다.

## 7. 향후 액션 아이템

### 즉시 실행 (우선순위 높음)

1. **네이버 Search Advisor 등록**
   - https://searchadvisor.naver.com 방문
   - minibolt.co.kr 사이트 추가
   - naver-site-verification 메타 태그로 소유권 확인 (이미 추가됨)
   - sitemap.xml 제출

2. **카카오 공유 캐시 초기화**
   - https://developers.kakao.com/tool/clear/og 방문
   - `https://minibolt.co.kr` 입력하여 캐시 초기화
   - 주요 제품 페이지 URL도 캐시 초기화

3. **네이버 블로그 개설 및 포스트 발행**
   - 네이버 블로그 개설 (미니볼트 공식 블로그)
   - 위 5개 포스트 순차 발행 (주 1~2회)
   - 각 포스트에 minibolt.co.kr 제품 페이지 링크 포함

### 중기 계획 (1~2주 내)

4. **제품 목록 페이지 SSR 전환**
   - `/products` 페이지를 Server Component 기반으로 리팩토링
   - 초기 제품 데이터를 서버에서 렌더링
   - 필터/검색 기능만 Client Component로 분리

5. **동적 OG 이미지 생성**
   - Next.js `ImageResponse` API를 사용하여 제품별 OG 이미지 자동 생성
   - 브랜드 로고 + 제품명 + 가격 + 규격 정보 포함

6. **네이버 스마트스토어 연동 검토**
   - 네이버 쇼핑 검색 노출을 위한 스마트스토어 개설 검토

### 장기 계획 (1~3개월)

7. **네이버 블로그 SEO 전략**
   - 주 2~3회 포스트 발행
   - 키워드 리서치 기반 콘텐츠 제작
   - 제품 활용 가이드, FAQ, 규격 비교 등

8. **네이버 검색광고 (파워링크) 검토**
   - 핵심 키워드별 CPC 분석
   - 테스트 캠페인 운영

9. **백링크 전략**
   - 메이커/DIY 커뮤니티 참여
   - 3D프린터, 드론 관련 포럼/카페에서 자연스러운 노출
